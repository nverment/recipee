# ReciPEE

A more complete app to serve as my personal recipe book.
So far, ReciPEE can:
1. Manage your recipes (Add, Remove and View - complete with ingredient list and instructions)
2. Search by recipe name or ingredients you want to use up
3. Manage ingredients

Future possible features:
- Edit recipse instead of having to create a new one each time
- Tag system (quick, spicy, vegan, etc.)
- UI?
- Cooking time
- Fuzzy search?
- Ratings - Comments
